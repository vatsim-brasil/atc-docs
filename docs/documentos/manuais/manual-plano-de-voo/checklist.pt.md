# Checklist rápido (FPL)

Use esta lista para evitar os erros que mais travam o fluxo (tanto em ATS real quanto em redes).

## Validação de consistência

- [ ] **ITEM 7** tem até 7 caracteres e bate com o callsign que você pretende usar
- [ ] **ITEM 8** (IFR/VFR) é coerente com sua rota (SID/STAR IFR etc.)
- [ ] **ITEM 9** usa designador OACI (Doc 8643). Se `ZZZZ`, existe `TYP/` no ITEM 18
- [ ] **ITEM 10** não promete equipamento que você não vai usar
- [ ] **ITEM 13** tem **ICAO 4 letras**. Se `ZZZZ`, existe `DEP/` no ITEM 18
- [ ] **ITEM 15** tem velocidade + nível coerentes (e VFR com `RMK/` quando necessário)
- [ ] **ITEM 16** tem destino + duração total (HHMM). Se `ZZZZ`, existe `DEST/` no ITEM 18
- [ ] Alternados coerentes (ou em branco conforme prática/regra). Se `ZZZZ`, existe `ALTN/`
- [ ] Se houver observação, use **`RMK/`** (evite texto solto sem indicador)

## Checklist operacional (simulação)

- [ ] METAR/TAF revisados (para selecionar pista e alternados)
- [ ] SID/STAR compatíveis com RNAV/PBN (se aplicável)
- [ ] Combustível planejado coerente com EET + reservas
- [ ] Transponder, altímetro, rádios e navegação configuráveis conforme o plano
